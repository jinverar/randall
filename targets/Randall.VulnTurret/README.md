# Randall.VulnTurret

Fictional **STT1** sentry-station lab — pan/tilt/track length-field crashes.

**Not** a weapon system: no fire control, arming, munitions, IFF, or real turret hardware. Loopback by default.

```bash
randall-vulnturret --mode tcp -p 15660
randall-vulnturret --mode udp -p 15661
```
